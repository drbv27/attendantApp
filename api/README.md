# Rutas para Users

Crear usuarios >>>>>METHOD POST
/api/users/create

Listar Todos los usuarios >>>>>METHOD GET
/api/users/all

Listar Usuario por Id >>>>>METHOD GET
/api/users/:id

Actualizar Usuario >>>>>METHOD PUT
/api/users/update

Eliminar usuario >>>>>METHOD DELETE
/api/users/delete/:id
